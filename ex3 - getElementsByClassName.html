getElementsByClassName() method

The getElementsByClassName() method returns a collection of all elements in the document with the specified class name, as an HTMLCollection object.

Tip: You can use the length property of the HTMLCollection object to determine the number of elements with the specified class name, then you can loop through all elements and extract the info you want.
