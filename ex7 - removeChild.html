HTML DOM removeChild() method

The removeChild() method removes a specified node of the specified element.

Returns the removed node as a Node object, or null if the node does not exist.

Note: The removed child is no longer part of the DOM. However, with the reference returned by this method, it's possible to insert the removed child to an element at a later time.

Tip: Use the appendChild() or insertBefore() method to insert the removed node into the same document.
